# Explicación de comandos básicos usados en los algoritmos de C# y Python

Console.ReadLine() -- Lee texto
int.Parse() -- Convierte <()> a un int

## Comandos usandos Python:

input() -- Lee texto
int() -- Convierte la cadena a un int
